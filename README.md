# Fundamentos del Aprendizaje Automático en Python

Bienvenido al curso de Fundamentos de Machine Learning. Este programa abarca desde el aprendizaje supervisado hasta la introducción al aprendizaje profundo y por refuerzo.

## Descripción
Aprende el arte del aprendizaje automático, predicción, reconocimiento de patrones y conceptos avanzados de Deep Learning.

## Contenido del Curso

1.  **Aprendizaje Supervisado con Scikit-learn**: Clasificación, regresión, ajuste de modelos y preprocesamiento.
2.  **Aprendizaje No Supervisado**: Clustering y reducción de dimensionalidad.
3.  **Introducción al Aprendizaje Profundo**: Conceptos básicos usando PyTorch.
4.  **Reinforcement Learning**: Uso de Gymnasium en Python.

## Proyectos Incluidos
*   Predictive Modeling for Agriculture
*   Clustering Antarctic Penguin Species
*   Taxi Route Optimization with Reinforcement Learning

## Configuración del Entorno

1.  **Crear el entorno virtual**:
    ```bash
    python3 -m venv .venv
    ```
2.  **Activar el entorno**:
    *   macOS/Linux: `source .venv/bin/activate`
    *   Windows: `.venv\Scripts\activate`
3.  **Instalar dependencias**:
    ```bash
    pip install -r requirements.txt
    ```

## Cómo Utilizar

Este proyecto utiliza **Quarto** para la documentación y presentación.

1.  Instala las dependencias (asegúrate de revisar `requirements.txt` si existe):
    ```bash
    pip install -r requirements.txt
    ```
2.  Visualiza el contenido:
    ```bash
    quarto preview
    ```
